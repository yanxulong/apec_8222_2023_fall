26 Intuitively, what is gradient descent?

  • We have a multidimensional parameter space. Calculate where the slope downwards is the steepest.

    • Traverse part of the way down that slope, then recalculate derivatives.

    • Iteratively do this until you find the bottom.

31 Neural networks: challenges

  • Throwing a bunch of nodes in layers won’t necessarily work well:

    • If you have lots of layers, it will take a long time to train

    • Simply adding more nodes doesn’t always increase predictive accuracy

  • Use problem knowledge to intelligently construct structure of network

    • What about problems that require very large data?

      • Maybe we want to impose the same weights in different parts e.g. “Convolutional Neural Networks”

34 Image analysis and CNNs.

Imagine an input image, where we’re trying to classify the image (rather than a pixel).

Say we have 3 bands for that image, and the image is 100x100 pixels. That’s 30,000 inputs.

Suppose we have one hidden layer with 10 neurons.

If we use a fully connected network (like in the code we just made), we would have 300,000 weights and 10 offsets to learn for that hidden layer. Too many.

36 Layer types in CNNs

  • Convolution layer:

    • Construct one or more “filters” that get passed over the pixel

    • Each filter has a window of say 5x5 pixels. It moves with some “stride” length over the image.

    • Need to chose activation function (sign, linear, logistic)
